How to Run The Program:
1. View "Project" files in the left pane
2. Right-click on lib > mysql-connector-java-8.0.30.jar and click on "Add As Library."
3. Leave the settings as default and click "Okay."
4. Add a new run configuration for the AppointmentManagerApplication class, name it whatever you like.
5. Ensure that IntelliJ finishes downloading the necessary dependencies.
6. Press the green play button at the top to begin!

Additional Report Description:
The additional report lets you view the number of appointments for each contact by week, month, or all-time.
This could be beneficial because we can see how meetings are being scheduled between contacts.

MySQL Connector Driver Version: mysql-connector-java-8.0.30
